Authentication

All API requests require authentication using an API key in the Authorization header.

HTTP Headers 
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
🔑 API Key

Get your API key from the Dashboard → Settings → API Keys

Send DM

Send a personalized DM to a single Twitter user.

POST /dms/send

Request Body

JSON 
{
  "recipient": "elonmusk",
  "template_id": "value_first",
  "variables": {
    "name": "Elon",
    "topic": "Mars colonization"
  },
  "personalization_level": 2,
  "track_opens": true
}

Response

200 OK 
{
  "success": true,
  "data": {
    "dm_id": "dm_123456789",
    "status": "sent",
    "recipient": "elonmusk",
    "message": "Hi Elon, I noticed your tweet about Mars colonization...",
    "personalization_score": 94,
    "cost": 0.015,
    "sent_at": "2024-03-05T14:30:00Z"
  }
}

Batch Send

Send DMs to multiple recipients at once (up to 1,000 per request).

POST /dms/batch
JSON 
{
  "recipients": ["user1", "user2", "user3"],
  "template_id": "networking",
  "personalization_level": 2,
  "schedule": {
    "start_time": "2024-03-06T09:00:00Z",
    "spread_hours": 8
  }
}

Get Templates

Retrieve all available DM templates or filter by category.

GET /templates

Query Parameters

Parameter Type Description
category string Filter by: cold, networking, sales, followup, recruiting
limit integer Max results (default: 20)

Personalize Message

Generate an AI-personalized message without sending.

POST /personalize
JSON 
{
  "recipient": "naval",
  "template_id": "value_first",
  "level": 2,
  "analyze_profile": true
}

Response

{
  "success": true,
  "data": {
    "message": "Hi Naval, I noticed your thread on wealth creation...",
    "personalization_score": 94,
    "profile_analysis": {
      "detected_interests": ["startups", "investing", "philosophy"],
      "recent_topics": ["wealth creation", "decision making"],
      "tone": "thoughtful"
    }
  }
}

🎯 Client Acquisition Integration

Seamlessly connect X-DM Pro with your client-acquisition pipeline.

Sync with Lead Scoring

Automatically sync high-scoring leads from your lead-scoring system to DM campaigns.

POST /integrations/leads/sync
JSON 
{
  "integration": "client-acquisition",
  "sync_config": {
    "min_lead_score": 75,
    "sync_frequency": "realtime",
    "auto_create_campaigns": true
  },
  "dm_settings": {
    "template_id": "value_first",
    "personalization_level": 2,
    "delay_after_lead": "24h"
  }
}

Lead Scoring Integration

Send DM engagement data back to your lead-scoring algorithm.

POST /integrations/leads/score 
{
  "lead_id": "lead_123",
  "dm_engagement": {
    "dm_opened": true,
    "dm_responded": true,
    "response_sentiment": "positive",
    "response_time_minutes": 45
  },
  "score_adjustment": {
    "dm_open": 5,
    "dm_response": 15,
    "positive_sentiment": 10
  }
}
💡 Tip

DM responses have shown to be strong buying intent signals. Recommend adding 15-25 points to lead score for DM responders.

Email Sequence Integration

Coordinate Twitter DMs with email sequences for multi-channel outreach.

POST /integrations/sequences/sync 
{
  "sequence_id": "seq_456",
  "sequence_steps": [
    {
      "order": 1,
      "channel": "twitter_dm",
      "template_id": "value_first",
      "delay_hours": 0
    },
    {
      "order": 2,
      "channel": "email",
      "template_id": "welcome_series",
      "delay_hours": 72,
      "condition": "if_no_response"
    },
    {
      "order": 3,
      "channel": "twitter_dm",
      "template_id": "follow_up",
      "delay_hours": 168
    }
  ]
}

Webhooks

Receive real-time notifications for DM events.

Configure Webhook

POST /webhooks/configure
{
  "url": "https://your-domain.com/webhooks/x-dm",
  "events": [
    "dm.sent",
    "dm.delivered",
    "dm.opened",
    "dm.responded",
    "lead.converted"
  ],
  "secret": "your_webhook_secret"
}

Webhook Payload Example

{
  "event": "dm.responded",
  "timestamp": "2024-03-05T14:35:00Z",
  "data": {
    "dm_id": "dm_123456",
    "campaign_id": "camp_789",
    "recipient": {
      "username": "sarahchen",
      "name": "Sarah Chen",
      "lead_score": 82
    },
    "response": {
      "message": "This looks interesting! Can you tell me more?",
      "sentiment": "positive",
      "response_time_seconds": 1800
    }
  }
}
