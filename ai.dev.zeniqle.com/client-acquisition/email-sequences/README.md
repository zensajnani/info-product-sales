# Email Sequence Templates

Conversion-optimized email sequences for different lead stages and personas.

## Sequences

1. **Welcome Series** - New subscriber onboarding
2. **Product Discovery** - Feature education for warm leads
3. **Case Study Nurture** - Social proof for consideration stage
4. **Pricing Conversation** - Objection handling and ROI focus
5. **Re-engagement** - Win back cold leads

## Usage

Each template includes:
- Subject line variants (A/B test ready)
- HTML and plain text versions
- Personalization tokens
- Send timing recommendations
- Performance benchmarks
